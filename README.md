# 🏛️ Nakhostin Academy Frontend

> **Minimal, High-Performance Static Web Application for Nakhostin Vocational Academy.**  
> Built with modern web technologies, optimized for local SEO, and containerized with Docker.

---

## 🛠️ Tech Stack

*   **Framework:** Next.js 15+ (App Router)
*   **Styling:** Tailwind CSS (Pure Dark Mode support)
*   **Analytics:** Umami Cloud (Privacy-focused tracking)
*   **Deployment:** Dockerized / Static Export (`output: 'export'`)
